# Day 2 - Data Structures in Java

## Implemented Structures
- QueueArray
- QueueLinkedList
- DoublyLinkedList
- CircularLinkedList

## How to Run
1. Navigate to the `src` directory.
2. Compile a file using `javac FileName.java`.
3. Run using `java FileName`.

Example:
```bash
javac QueueArray.java
java QueueArray
```
